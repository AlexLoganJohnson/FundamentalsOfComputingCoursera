integer: is a whole number (3, 1, 2, etc.)
float: is a decimal number, including 3.0

python uses about 15 decimal points

Arithmetic Operators:
+  Plus        Addition
-  Minus       Subtraction
*  Times       Multiplication
/  Divided by  Divison
** Power       Exponentiation

If both operands are integers python returns the answer as an integer
e.g. print (1/3, 5/2, -7/3)
0 2 -3

Python has operand precedence according to PEMDAS
